Hydra

Distributed Data Collection and Warehousing

Hydra is a distributed web application designed for generalized data collection and warehousing. Designed from the ground-up to maximize portability, the core doesn't rely on any specific Cloud platform. You can self-host it, host it on a VPS, or host it on Amazon Web Services equally well. The base system comes with a File Explorer interface for easily uploading, finding, and manipulating binary files stored to the system, and the common REST API can be reutilized to implement any kind of custom implementation specific to your domain, such as browser plugins, webapps, and native mobile applications. Hydra is open source under the GNU General Public License v3, and the source materials are available for download from the Mershon Enterprises GitHub.com account. WH Hero

Interactive data collection and visualization of WirelessHART mesh data

WH Hero is a mobile application developed with the Ionic Framework and HART-IP, allowing you to easily connect to a WirelessHART Gateway and quickly interact with and visualize the collective dataset of a WirelessHART wireless mesh. WH Hero is available now on Google Play, and the Apple App Store. WH Hero is open source under the GNU General Public License v3, and the source materials are available for download from the Mershon Enterprises GitHub.com account.

Hydra

Distributed Data Collection and Warehousing

Hydra is a distributed web application designed for generalized data collection and warehousing. Designed from the ground-up to maximize portability, the core doesn't rely on any specific Cloud platform. You can self-host it, host it on a VPS, or host it on Amazon Web Services equally well. The base system comes with a File Explorer interface for easily uploading, finding, and manipulating binary files stored to the system, and the common REST API can be reutilized to implement any kind of custom implementation specific to your domain, such as browser plugins, webapps, and native mobile applications.
